# Notebooks

Suggested notebook sequence:

1. `01_data_audit.ipynb` - load the raw CSV, check row counts, missing values, category labels, and randomization balance.
2. `02_experiment_analysis.ipynb` - compare control and treatment groups on visit, conversion, and spend.
3. `03_segment_uplift_analysis.ipynb` - explore treatment effects by customer segment.
4. `04_decision_memo_figures.ipynb` - create final portfolio visuals for the decision memo.

Keep notebooks readable. Each notebook should answer one stage of the project rather than becoming one giant scratchpad.
